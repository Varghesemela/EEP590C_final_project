\doxysubsection{Detailed Description}
Global variables, constants, and peripheral instances for the Smart Security Door System. 

This file holds the definitions for globally shared variables, task handles, hardware pin configurations, Free\+RTOS synchronization primitives, queues, and sensor state flags. These definitions are used across multiple modules including sensor processing, RFID access control, LCD display handling, and timing logic.

Peripheral Components\+:
\begin{DoxyItemize}
\item Ultrasonic Sensor (TRIG, ECHO)
\item PIR Motion Sensor
\item RFID Reader (MFRC522)
\item I2\+C-\/based LCD
\item Servo Motor
\item RTC (DS3231)
\end{DoxyItemize}

System Synchronization\+:
\begin{DoxyItemize}
\item Free\+RTOS tasks and task handles
\item Queues for sensor data and RFID UIDs
\item Semaphores for I2C access
\item Timers for backlight and lock control
\end{DoxyItemize}
