\hypertarget{_e_e_p590_c__final__project_8ino_overview}{}\doxysubsection{\texorpdfstring{Overview}{Overview}}\label{_e_e_p590_c__final__project_8ino_overview}
This sketch sets up all system resources and tasks needed for a smart security and monitoring system implemented on ESP32. It includes I2C, RFID, servo control, LCD display, RTC setup, and Free\+RTOS-\/based task management.
