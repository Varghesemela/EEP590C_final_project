Detailed Description

FreeRTOS-based Light Monitoring and Anomaly Detection System using ESP32.

Overview

This sketch sets up all system resources and tasks needed for a smart security and monitoring system implemented on ESP32. It includes I2C, RFID, servo control, LCD display, RTC setup, and FreeRTOS-based task management. Timers and semaphores are used to coordinate access to shared resources.
